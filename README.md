# VTT Frontend (En Desarrollo)

Frontend del sistema Virtual Tabletop (VTT), implementado con React y Vite.
Este repositorio define la interfaz de usuario, la interacción con la API y la experiencia visual del juego.
Actualmente se encuentra en desarrollo activo, por lo que la estructura, componentes y vistas pueden cambiar con frecuencia.

---

# Estado del Proyecto

Este frontend está en construcción.
Se están creando los componentes iniciales, hooks genéricos y la conexión con la API del backend.
La interfaz final, rutas estables y manejo completo de la interacción con el backend se añadirán conforme avance el desarrollo.
